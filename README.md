# ReCo (Repository of Configurations)

📑 Draft version (RU):
ReCo — это централизованный репозиторий конфигураций Windows 11, созданных для различных сценариев использования.

📑 Poject info (Global):

The project was initially created with a complex modularity method, which was convenient and easy for the Author to understand. Currently, everything fits into the playbook format (.apbx) for AME. If you have questions or proposals regarding licenses and/or other issues, write to the Author on Telegram.

⚠️ Attention:
We are currently using a temporary logo for this project while we finalize its style and color palette. Please note that the current design shares similarities with the ReviOS logo. Should the ReviOS authors express any concerns or dissatisfaction, this prototype will be immediately removed. We hold fellow creators in high regard and have no intention of infringing upon their work.
